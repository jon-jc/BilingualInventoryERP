# Bilingual Inventory ERP

This is a mini ERP (Enterprise Resource Planning) system focused on inventory management. It's built using .NET Core and Windows Forms, with bilingual support for English and Japanese.

## Features

- Product management (Add, Update, Delete)
- Bilingual interface (English/Japanese)
- SQL Server database integration

## Getting Started

1. Clone the repository
2. Open the solution in Visual Studio or VS Code
3. Update the connection string in `Form1.cs` to point to your SQL Server
4. Run the application
